# JAMIESSHOESS — Brand Package

This folder holds the official brand assets from designer **Tyler Crowe** (delivered Dec 2025).

## Folder structure

```
brand/
├── logos/           — Primary logos, "Jamies" abbreviated mark, slogan marks
│                      Formats: .svg (preferred for web), .png (transparent), .jpg (black bg)
│
├── illustrations/   — Walking man (mascot), Standing guy, t-shirt graphics
│
├── stickers/        — Badge formats, sticker designs
│
├── lockups/         — Round, rectangular, wide combined graphics (logo + illustration)
│
└── colors.json      — Brand color hex values + usage notes
```

## How to use this folder

1. **Drop Tyler's files into the matching subfolder** when they arrive
2. **Reference them in HTML/CSS** using relative paths: `brand/logos/jamiesshoess-primary.svg`
3. **Prefer SVG** for any logo or illustration that needs to scale (the site uses these at multiple sizes)
4. **Use PNG** when the asset needs transparency over a non-black background
5. **Use JPG (black BG)** for social-style sections only

## Brand identity notes from Tyler

- **Aesthetic:** Hand-painted, distressed, graffiti-influenced. NOT clean/corporate. Intentional imperfection.
- **Distortion:** Logos and graphics have intentional distortion to feel hand-painted, not digital.
- **Drop shadows + outlines** on primary logo for depth.
- **Halftone gradients** used with the green color — use sparingly.

## Color usage

- **Mint** — primary teal/mint (heavy hitter, use frequently)
- **Flash** — second primary (heavy hitter, use frequently)
- **Cash** — green, used sparingly, primarily as halftone gradient
- **Background** — black is primary
